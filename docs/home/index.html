Welcome

Welcome to the Albert launcher documentation. Here you should find anything you need to know about Albert.

The user guide should get you started. If you want to take part the Developer guide will help you getting in. The Troubleshooting helps solving common problems. The section State of the project shows where we are an where to go. Finally the section Contact tells you how and when to contact me/us.
