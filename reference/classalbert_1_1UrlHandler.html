Detailed Description

Albert scheme URL handler interface.

Use this interface to register albert: URL handlers based on Extension::id. URLs with the host matching this extension's id are passed to the handle() method. E.g. the URL albert://github/?â€¦ will be redirected to the GitHub extension.
